Algorithm Visualizer

An interactive tool that visually demonstrates how different algorithms work in real time. Users can input a starting point, and the visualizer will showcase how popular algorithms (i.e. BFS or DFS) traverse a 2-D grid.

This project provides an interactive way for users to understand and learn about algorithms.

Task Manager

A program designed to assist users in managing their tasks effectively. Users can create, prioritize, and delete their own tasks accordingly.

The program utilizes Java Oriented Programming to create objects represented as a user defined task.
