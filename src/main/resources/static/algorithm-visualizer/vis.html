Welcome to the Algorithm Visualizer! visualize algorithms in real time.

The application provides a grid-based environment where you can navigate a sprite, customize blocked paths, and witness the real-time visualization of two powerful search algorithms: Breadth-First Search (BFS) and Depth-First Search (DFS).


How to Use

  • Drag and drop the sprite across the grid to set its initial position.
  • Select an algorithm from the dropdown menu.
  • Influence the algorithm's pathfinding behavior by creating blocked paths. Simply click on grid cells to customize your environment.
  • Click the "Visualize" button to see the algorithm adapting to the obstacles you've set.
  • After the visualization is complete, click the 'Reset' button to start fresh.


Note

Currently only supports BFS (Breadth-first-search) and DFS (Depth-first-search) algorithms. Cannot currently drag-n-drop on Mobile Devices :/
