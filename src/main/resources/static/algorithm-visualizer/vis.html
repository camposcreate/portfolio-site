Welcome to the Algorithm Visualizer! visualize algorithms in real time.


Witness the visualization of two powerful search algorithms: Breadth-First Search (BFS) and Depth-First Search (DFS)!


Learning Objective

Provide a user-friendly platform for exploring search algorithms designed to find the shortest path between two points. Featuring two widely-used algorithms, BFS and DFS, this application illustrates their execution on a grid. Users can gain a comprehensive understanding of these algorithms' mechanics and their crucial role in pathfinding.


How to Use

  • Drag and drop the sprites across the grid to set their initial and ending position.
  • Select an algorithm from the dropdown menu.
  • Influence the algorithm's pathfinding behavior by creating obstacles. Simply click on open grid cells below to customize your environment.
  • Click the 'Visualize' button to see the algorithm adapting to the obstacles you've set.
  • After the visualization is complete, click the 'Reset' button to start fresh.
