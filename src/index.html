Algorithm Visualizer

The Algorithm Visualizer project is an interactive tool that visually demonstrates how different algorithms work in real time. Users can input a starting point and an ending point, and the visualizer will showcase how algorithms such as Dijkstra's algorithm or A* algorithm find the shortest path between the two points. This project provides an intuitive and interactive way for users to understand and learn about algorithms.
